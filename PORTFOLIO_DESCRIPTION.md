## CareerPilot AI — Job Application Tracker & CV Match Analyzer

**Tech:** Python, Streamlit, SQLite, pandas, Plotly, scikit-learn, pytest, Docker, GitHub Actions

Built a local-first job-search command center that tracks applications, priorities, deadlines, CV versions, and pipeline status. Implemented an explainable CV-to-job matcher using TF-IDF, cosine similarity, and deterministic skill-gap analysis. Added source-performance analytics, CSV export, input validation, automated tests, CI across multiple Python versions, and containerized deployment.
